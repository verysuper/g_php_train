淺談 BASE64 編碼與線上資源

Base64 編碼的使用在網頁設計與資料庫處理中是一項很基本卻很實用的技術，在不考慮安全性的情況下，也可以當做是最基礎的「文字編碼」處理。Base64 只用到可列印的 62+2 個字元來傳輸資料，因此可以將「中文字」或「二元檔(Binary File)」轉換之後以純文字的方式進行傳輸。

Base64 編碼的原理、介紹、官方文件及其他相關資源，請參考維基百科的說明：

  • Base64 - 維基百科
    http://zh.wikipedia.org/w/index.php?title=Base64&variant=zh-tw

在 PHP 程式裡已經有提供了現在的函數可供文字處理，不需要再另行撰寫 Base64 的函數，如：

  • 編碼：base64_encode()
    http://tw.php.net/manual/en/function.base64-encode.php
    http://stock.ettoday.com/book/function.php-base64_encode.htm
  • 解碼：base64_decode()
    http://tw.php.net/manual/en/function.base64-decode.php
    http://stock.ettoday.com/book/function.php-base64_decode.htm

直角在這邊要介紹的是底下這個網站，線上的 Base64 編/解碼程式：

  • Base64 Online - base64 decode and encode
    http://www.motobit.com/util/base64-decoder-encoder.asp

這個網站除了可以自行輸入文字轉換之外，還可以直接輸入一個檔案喔！不但如此，連二元檔都可以上傳轉換成 Base64 的編碼。當然，最神奇的是，你可以輸入已經編碼過二元檔的編碼字串，它會幫你還原成為二元檔喔！

  • What to do with the source data:
    指的是你要工作的項目，encode 代表編碼、decode 代表解碼。
  • Output data:
    output to a textbox (as a string) → 轉換為文字格式。
    export to a binary file, filename: → 轉換為二進位檔案，需輸入檔名。
  • 文字區
    Base64 representation of the source data: → 已經編碼過的 Base64 文字。
    Type (or copy-paste) some text to a textbox bellow. → 代表未編碼的原始文字。
